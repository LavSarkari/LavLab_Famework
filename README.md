# LavLab_Famework
A Small Famework for noob hackers....
# Installation
*For Termux*
```
apt install git -y
apt install python -y
git clone https://github.com/LavSarkari/LavLab_Famework/
cd LavLab_Famework
bash setup.sh
```
*For Linux*
```
apt install git -y
apt install python -y
git clone https://github.com/LavSarkari/LavLab_Famework/
cd LavLab_Famework
python LavLab.py
```
